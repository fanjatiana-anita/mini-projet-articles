PHASE 5 — URL REWRITING (Anita- ~1h)
================================================================================

[ ] 5.1 Créer le fichier .htaccess à la racine du FrontOffice
        Contenu :
        ────────────────────────────────────────
        Options +FollowSymLinks
        RewriteEngine On

        # Réécriture page article
        RewriteRule ^article/([a-z0-9\-]+)/?$ article.php?slug=$1 [L,QSA]

        # Réécriture catégorie
        RewriteRule ^categorie/([a-z0-9\-]+)/?$ categorie.php?slug=$1 [L,QSA]

        # Réécriture page simple
        RewriteRule ^page/([a-z0-9\-]+)/?$ page.php?slug=$1 [L,QSA]
        ────────────────────────────────────────

[ ] 5.2 Activer mod_rewrite dans le Dockerfile
        - RUN a2enmod rewrite
        - Dans apache config : AllowOverride All

[ ] 5.3 Tester les URLs réécrites
        - http://localhost:8080/article/guerre-iran → doit fonctionner
        - http://localhost:8080/categorie/actualites → doit fonctionner

[ ] 5.4 Générer automatiquement les slugs depuis le titre

PHASE 6 — SEO - CHECKLIST COMPLÈTE (Olivia - ~1h)
================================================================================

[ ] 6.1 Balises meta sur TOUTES les pages
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta name="description" content="...max 160 caractères...">
        <meta name="keywords" content="guerre iran, conflit iran, ...">
        <title>Titre de la page | Nom du site</title>

[ ] 6.2 Structure des titres (OBLIGATOIRE)
        - 1 seul <h1> par page
        - <h2> pour les sections principales
        - <h3>, <h4> pour les sous-sections
        - NE PAS sauter de niveau (h1 → h3 est interdit)

[ ] 6.3 Images
        - Chaque <img> doit avoir un attribut alt descriptif
        - Exemple : alt="Soldats iraniens lors d'un exercice militaire 2024"
        - Nommer les fichiers images avec des tirets (pas d'espaces)

[ ] 6.4 Autres bonnes pratiques SEO
        - URLs courtes et lisibles (grâce au rewriting)
        - Pas de contenu dupliqué
        - Texte lisible (pas de Lorem Ipsum en production)
        - Liens internes entre articles
